## Packages
date-fns | Used for robust date manipulation and formatting
japanese-holidays | Essential for calculating Japanese public holidays accurately
framer-motion | For smooth page transitions and micro-interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging tailwind classes

## Notes
- LocalStorage is used as the primary data store as requested. No external API calls are made for work logs.
- Assumes browser environment supports Japanese date formatting (`ja` locale in date-fns).
